# See All

See all is a web application that allows users to analyze and transcribe live video streams using AI services.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

Docker and Docker Compose are required to run the project.

### Installing

1. Clone the repository:
   ```bash
   git clone https://github.com/rajahafify/see-all.git
   ```

## Running the Project

1. Start the project:
   ```bash
   docker-compose up --build
   ```
